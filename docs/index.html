Features

Everything you need to land the role

A complete AI pipeline — from discovery to offer — with you in control of every decision.

🎯
A-F Evaluation Engine
10 weighted dimensions — match, gaps, comp research, STAR stories — scored per archetype.
📄
ATS PDF Generation
Keyword-injected, tailored CVs with Space Grotesk + DM Sans design via Playwright.
✉️
Cover Letter Generator
Research-backed letters with keyword mirroring and four interactive angle prompts. PDF-ready.
🔍
Portal Scanner
45+ companies pre-configured across Greenhouse, Ashby, Lever, Wellfound, and more.
⚡
Batch Processing
Evaluate 10+ offers in parallel with headless CLI workers. Sub-agents do the heavy lifting.
📖
Interview Story Bank
STAR+Reflection stories accumulate across evaluations — 5–10 master answers for any question.
💰
Negotiation Scripts
Salary frameworks, geographic discount pushback, and competing-offer leverage templates.
📊
Dashboard TUI
Go + Bubble Tea terminal UI — 6 filter tabs, 4 sort modes, inline status changes. Catppuccin Mocha theme.
🛡️
Human-in-the-Loop
AI evaluates and recommends. You decide and act. The system never auto-submits an application.

Commands

One command, fourteen modes

Paste a job URL or description directly — career-ops auto-detects and runs the full pipeline.

career-ops commands
$/career-ops {paste a JD}— full pipeline
$/career-ops scan— scan portals
$/career-ops pdf— generate CV
$/career-ops cover— cover letter
$/career-ops batch— batch evaluate
$/career-ops tracker— view pipeline
$/career-ops deep— company research
$/career-ops contacto— LinkedIn outreach

Quick Start

Up and running in minutes

Works with Claude Code, OpenCode, Codex, Qwen, Antigravity CLI, and GitHub Copilot.

1

Install with one command

Run npx @santifer/career-ops init — no global install needed. Requires Node.js.

2

Open your AI CLI

Navigate into career-ops/ and run claude, opencode, codex, or any supported CLI.

3

Get onboarded by the AI

On first launch, career-ops walks you through setup — your CV, profile, and target roles — just by chatting.

4

Start evaluating

Paste a job URL or description. Get an A-F score, tailored CV, and cover letter — all in one go.


$npx @santifer/career-ops init
$cd career-ops && claude
